## вңҰ Overview

**Lattice** is a production-grade AI platform that conducts and evaluates technical interviews in real time. It unifies a low-latency **Voice Agent**, **Computer Vision**, and **Deep Audio Analysis** into a single end-to-end pipeline вҖ” turning a raw interview session into a structured, scored report.

> Candidates are assessed across **12 behavioral dimensions** using a tri-modal signal: what they *say*, how they *sound*, and how they *appear*.

## рҹҡҖ Quick Start

### Prerequisites

- рҹҗі **Docker Desktop** вҖ” running
- рҹҗҚ **Conda** вҖ” `pupil310` environment active
- рҹҺһпёҸ **FFmpeg** вҖ” on system `PATH`

### Launch

```powershell
# Clone the repo
git clone https://github.com/your-org/lattice.git
cd lattice

# Fire everything up
.\start-system.ps1
```

The orchestration script brings up all services automatically:

```
вң”  Redis        вҖ” signaling store
вң”  LiveKit      вҖ” WebRTC SFU
вң”  Vision       вҖ” port 8000
вң”  Inference    вҖ” port 8001
вң”  Celery       вҖ” task workers
вң”  Next.js      вҖ” port 3000  вҶҗ open this
```

> Visit **`http://localhost:3000`** to start an interview session.
